Primeiros Passos com HTML

Neste curso aprenderemos os primeiros passos em HTML.

O curso está dividido em 04 módulos, com fácil compreenção.

Bons estudos!

Tópicos deste curso:

  • Módulo 01 - Introdução ao HTML;
  • Módulo 02 - Estrutura de uma página HTML;
  • Módulo 03 - Elementos avançados do HTML;
  • Módulo 04 - Interatividade e animações em HTML;

Módulo 01 - Introdução ao HTML

Este módulo irá introduzir os conceitos básicos do HTML, incluindo a estrutura básica de uma página HTML e as tags mais comuns utilizadas no desenvolvimento web.

Módulo 02 - Estilizando uma página HTML

Este módulo irá ensinar como adicionar estilos a uma página HTML utilizando CSS, incluindo a aplicação de cores, fontes, tamanhos e posicionamento dos elementos.

Módulo 03 - Elementos avançados do HTML

Este módulo irá explorar elementos mais avançados do HTML, como formulários, tabelas e elementos multimídia, permitindo o desenvolvimento de páginas mais complexas.

Módulo 04 - Interatividade e animações em HTML

Este módulo irá abordar como adicionar interatividade e elementos animados a uma página HTML, utilizando JavaScript e bibliotecas populares de animação, como o JQuery.
